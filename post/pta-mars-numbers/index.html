PTA Mars Numbers

2019-12-06 / 1 min read

题目

People on Mars count their numbers with base 13:

Zero on Earth is called "tret" on Mars.
The numbers 1 to 12 on Earth is called "jan, feb, mar, apr, may, jun, jly, aug, sep, oct, nov, dec" on Mars, respectively.
For the next higher digit, Mars people name the 12 numbers as "tam, hel, maa, huh, tou, kes, hei, elo, syy, lok, mer, jou", respectively.
For examples, the number 29 on Earth is called "hel mar" on Mars; and "elo nov" on Mars corresponds to 115 on Earth. In order to help communication between people from these two planets, you are supposed to write a program for mutual translation between Earth and Mars number systems.

Input Specification:
Each input file contains one test case. For each case, the first line contains a positive integer N (<100). Then N lines follow, each contains a number in [0, 169), given either in the form of an Earth number, or that of Mars.

Output Specification:
For each number, print in a line the corresponding number in the other language.

Sample Input:
4
29
5
elo nov
tam

Sample Output:
hel mar
may
115
13

代码不难，需要注意一下几点：

  • 那个数组，一定一定不要打错了！
  • 13进制，不是12进制！
  • 需要注意13 26 39只需要输出高位即可，低位的0不需要输出
  • 关于判断数字，不要拿长度啊，111的长度也是3呀


#include <iostream>
#include <string>
#include <vector>
using namespace std;

string high[12] = { "tam", "hel", "maa", "huh", "tou", "kes", "hei", "elo", "syy", "lok", "mer", "jou" };
string low[13] = { "tret", "jan", "feb", "mar", "apr", "may", "jun", "jly", "aug", "sep", "oct", "nov", "dec" };

string change(string ori)
{
	string res="";
	int sum = 0;
	// 若是数字
	if (ori[0]<='9' && ori[0]>='0')
	{
		int num = atoi(ori.c_str());
		int h = num / 13 - 1;  // 是13进制不是12进制
		int l = num % 13;
		if (h == -1)
			res += low[l];
		else
		{
			if (num % 13 != 0) {
				res.append(high[h]);
				res.append(" ");
				res.append(low[l]);
			}
			else
				res.append(high[h]);
		}
	}

	// 若是火星文
	if(ori.size() >= 3)
	{
		// 如果只有3位
		if (ori.size() <= 4) {
			for (int i = 0; i < 13; i++)
			{
				if (ori == low[i])
					res.append(to_string(i));
			}
			// 只有高位，是乘，不是加
			if (res == "")
			{
				for (int i = 0; i < 12; i++)
				{
					if (ori == high[i])
						res.append(to_string((i+1) * 13));  // 都说是13进制啦
				}
			}
		}
		else {
			string hh, ll;
			hh = ori.substr(0, 3);
			ll = ori.substr(4, 3);
			for (int i = 0; i < 12; i++)
				if (hh == high[i])
					sum += (i + 1) * 13;
			for (int i = 0; i < 13; i++)
				if (ll == low[i])
					sum += i;
			res = to_string(sum); 
		}
	}
	//cout << res << endl;
	return res;
}

int main() {
	int n = 0;
	string input[100];
	cin >> n;
	getchar();
	for (int i = 0; i < n; i++)
		getline(cin, input[i]);
	for (int i = 0; i < n; i++)
		cout << change(input[i]) << endl;

	system("pause");
}
